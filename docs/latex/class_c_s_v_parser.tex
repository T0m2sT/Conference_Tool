Parse the CSV file and fill the given containers with data. 

Reads the file line by line, detects sections (\+\#\+Submissions, \+\#\+Reviewers, \doxylink{struct_parameters}{Parameters}, \doxylink{struct_control}{Control}), and delegates each data line to parse\+Line().

{\bfseries{Time Complexity:\+}} O(L · T) where L = number of lines in the file, T = max tokens per line. In practice O(\+L) since T is bounded by the fixed CSV column count.

{\bfseries{Space Complexity:\+}} O(\texorpdfstring{$\vert$}{|}\+S\texorpdfstring{$\vert$}{|} + \texorpdfstring{$\vert$}{|}\+R\texorpdfstring{$\vert$}{|}) for the parsed submissions and reviewers.
