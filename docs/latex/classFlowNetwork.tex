\doxysubsection{Detailed Description}
Builds a Max-\/\+Flow network from conference data and solves the review assignment problem. 

Node naming scheme (used as vertex info in \doxylink{classGraph}{Graph$<$std::\+string$>$}):\+ "{}source"{} — the super source "{}sink"{} — the super sink "{}sub\+\_\+$<$id$>$"{} — submission node "{}rev\+\_\+$<$id$>$"{} — reviewer node

Time complexity of build\+And\+Solve:\+ O(\+V \texorpdfstring{$\ast$}{*} E\texorpdfstring{$^\wedge$}{\string^}2) for Edmonds-\/\+Karp, where V = 2 + \texorpdfstring{$\vert$}{|}submissions\texorpdfstring{$\vert$}{|} + \texorpdfstring{$\vert$}{|}reviewers\texorpdfstring{$\vert$}{|}, E = edges based on domain matching.
