\doxysubsection{Detailed Description}
Converts domain data into display-\/ready string lines. 

Responsible for formatting submissions, reviewers, parameters, and assignment results into vectors of strings that the \doxylink{classMenu}{Menu} can render. Also provides text wrapping utilities for display formatting.
